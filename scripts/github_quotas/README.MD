# get github quotas
This script allows you to get notifications to telegram about your github account quotas


## Installation:

 Requirements

     Python 3.x
     requests
     load_dotenv
     json


## How to run:

1. Obtain a Github developer token.

2. Create .env file and add your tokens as environment variables:
   ```
     GITHUB_TOKEN=
     GITHUB_USER=
     #telegram
     BOT_TOKEN=
     BOT_CHAT_ID=
   ```
